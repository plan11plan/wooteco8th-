# java-convenience-store-precourse


# 기능요구사항
- 온보딩 메세지를 출력한다.
- 12월 식당 예상 방문 날짜를 입력받는다.
    - 유효한 날짜 형식이 아니면 -> 예외 메세지
- 주문을 입력 받는다.
    - 유효한 주문 형식이 아니면 -> 예외메세지
- 손님 방문날짜와 함께 이벤트 혜택 안내 메세지를 출력한다.
- 손님이 주문한 메뉴정보를 출력한다.

### 계산 
- 할인 전 총주문 금액을 계산하여 출력한다.
- 증정 메뉴정보를 계산하여 출력한다.
- 혜택 내역을 계산하여 출력한다.
- 총 혜택 금액을 계산하여 출력한다.
- 할인 후 예상 결제 금액을 계산하여출력한다.
- 12월 이벤트 배지를 계산하여 출력한다.
